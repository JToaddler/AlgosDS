	Repeat all above three in iterative mode


InOrder traversal
	First, visit all the nodes in the left subtree
	Then the root node
	Visit all the nodes in the right subtree
	
PreOrder traversal
	Visit root node
	Visit all the nodes in the left subtree
	Visit all the nodes in the right subtree

PostOrder traversal
	Visit all the nodes in the left subtree
	Visit all the nodes in the right subtree
	Visit the root node
